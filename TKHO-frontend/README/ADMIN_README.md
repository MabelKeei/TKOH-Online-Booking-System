# Admin Management System

## 功能概述

管理员系统包含以下6个核心功能模块：

### 1. 车位管理 (Parking Management)
- 添加、编辑、删除停车位
- 管理车位类型（EV充电位/普通车位）
- 设置车位状态（启用/禁用）
- 车位位置信息管理

### 2. 房间管理 (Venue Management)
- 添加、编辑、删除会议室和场地
- 上传场地图片（支持多图）
- 设置场地类型（会议室/其他场地）
- 管理场地容量和位置信息
- 场地状态管理

### 3. 员工账号管理 (Employee Management)
- **员工列表**：
  - 查看所有员工信息
  - 编辑员工资料
  - 管理员工年度预定配额
  - 重置员工配额
  - 删除员工账号

- **待审批员工**：
  - 新员工扫码注册后进入待审批列表
  - 管理员审批员工信息
  - 审批通过后自动创建账号
  - 可拒绝不符合要求的注册

- **二维码注册**：
  - 生成员工注册二维码
  - 新员工扫码填写个人信息
  - 提交后等待管理员审批

### 4. 权限管理 (Permission Management)
- 创建和管理职位/角色
- 为每个角色设置年度预定配额
- 查看每个角色的员工数量
- 编辑角色描述和权限
- 删除未分配员工的角色

### 5. 会议审批 (Meeting Approval)
- **待审批**：
  - 查看所有待审批的会议预定
  - 审批会议名称和预定信息
  - 批准或拒绝预定请求
  - 拒绝时需填写原因

- **已批准**：
  - 查看所有已批准的会议记录
  - 显示审批时间和审批人

- **已拒绝**：
  - 查看所有被拒绝的预定
  - 显示拒绝原因

**注意**：
- 只有预定房间需要审批
- 预定EV车位不需要审批
- 房间预定提交后自动占用时间段
- 管理员只需审批会议名称是否合适

### 6. 系统提示词管理 (System Prompts)
- 管理系统各类提示消息
- 支持的消息类型：
  - Success（成功）
  - Warning（警告）
  - Error（错误）
  - Info（信息）
- 编辑提示消息内容
- 启用/禁用特定提示
- 支持变量占位符（如 {reason}）

## 访问方式

1. 登录页面选择 "Admin Management"
2. 使用管理员账号登录（默认：test / 123456）
3. 进入管理员主界面
4. 点击对应功能卡片进入管理页面

## 技术实现

- 前端框架：Vue 3 + Element Plus
- 路由：Vue Router
- 状态管理：Pinia
- API接口：RESTful API（见 `/src/api/admin.js`）
- 组件位置：`/src/components/admin/`

- `PromptManagement.vue` - 系统提示词管理

## 数据流程

### 员工注册流程
1. 新员工扫描二维码
2. 填写个人信息（Corp ID、姓名、部门、职位、邮箱）
3. 提交后进入待审批列表
4. 管理员审批通过
5. 系统自动创建账号并分配默认配额

### 会议预定审批流程
1. 员工提交会议室预定
2. 系统自动占用时间段
3. 预定进入待审批列表
4. 管理员审批会议名称
5. 批准后员工收到通知
6. 拒绝后释放时间段并通知员工

## 配额管理

- 每个员工有年度预定配额
- 配额由员工的职位/角色决定
- 管理员可以单独调整员工配额
- 管理员可以重置员工已使用配额
- 配额用完后无法继续预定
